# Australia's Demographic Transformation (1996–2016)

## Overview
Analysis of 20 years of ABS census data to uncover migration-driven population changes across Australian states and key Asia-Pacific source countries.

## Business problem
Policymakers and urban planners lack clear, data-driven visibility into how migration patterns are reshaping Australia's workforce composition, settlement geography, and long-term economic structure.

## Tools & techniques
- Power BI (3 interactive dashboards, DAX measures, slicers)
- Tableau (supplementary visualisation)
- Python / Excel (data cleaning, filtering 256 → 10 key countries)

## Key findings
- China was the fastest-growing migrant group (+370%, 118K → 558K)
- NSW absorbed ~50% of Chinese migrants — driving housing & infrastructure pressure
- Working-age migrants (25–64) dominate across all source countries — migration is economically targeted
- Gender composition varies by origin: China/Philippines female-majority (education/care), India/Sri Lanka male-majority (skilled labour)

## Dashboards
1. Population overview — growth trends, age-sex structure, state distribution
2. Demographic comparison — cross-country migration patterns by region and gender
3. Chinese migration case study — deep-dive into the fastest-growing cohort
